# Terraform AWS Minecraft Full Example

This project is an example of how to use the [terraform-aws-minecraft](../..) module.

## Deployment

### Prerequisites

- [Terraform](https://www.terraform.io/downloads.html)

### Bootstrap

```sh
terraform init
```

### Validate and apply

```sh
terraform plan
terraform apply
```
